## Common Questions

**Q: Page looks plain?**
A: Make sure styles.css is loading. Check:
   - styles.css exists in root folder
   - File is named exactly "styles.css" (lowercase)
   - No typos in index.html link

**Q: Navigation doesn't work?**
A: Check file names match exactly:
   - index.html âœ“
   - pages/about.html âœ“
   - pages/blog.html âœ“
   - pages/contact.html âœ“

**Q: Images don't show?**
A: Add images to assets/ folder:
   - Right-click assets folder
   - "Add new file" or paste image
   - File shows as ../assets/imagename.jpg

**Q: Contact form doesn't send?**
A: Need to setup Formspree (optional):
   - Go to formspree.io
   - Sign up free
   - Create form
   - Copy your ID
   - Paste in contact.html form tag
   - See GIT_SETUP.md for details
